# Splunk Journey

A hands-on Splunk learning journey from beginner to SOC analyst.

## About

This repository documents my day-by-day progress while learning Splunk through hands-on practice, lab exercises, and real-world scenarios.

The goal of this journey is to develop practical skills in:

* Splunk Enterprise
* Log Analysis
* Search Processing Language (SPL)
* Dashboards and Reports
* Alerts and Monitoring
* Security Operations (SOC)
* Threat Hunting
* Detection Engineering

## Lab Environment

### Host Machine

* OS: Xubuntu

### Virtualization

* QEMU/KVM

### Guest Machine

* Debian Minimal

### SIEM Platform

* Splunk Enterprise
